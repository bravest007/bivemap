# 2. 핵심 기능 명세

[<- 마스터 인덱스로 돌아가기](./index.md)

이 시스템의 뼈대를 이루는 4가지 핵심 MVP 기능 명세입니다.

### [기능 1] 매직 날리지 주입기 (Magic Ingestion & Auto-Translation)
- **설명**: 텍스트, URL, 로컬 파일(Drag & Drop) 주입 시 백그라운드 AI 파서가 작동합니다. **특히 영어 등 외국어 원문 자료라도 100% 한글로 자동 번역 및 요약**해 주며, 핵심 영문 용어만 보존하여 기존 맵에 적절히 자동 배치합니다.
- **사용자 시나리오**: X(트위터)에서 온통 영어로 적힌 긴 최신 AI 논문 기사를 마인드맵 빈 곳으로 냅다 드래그 하세요. 시스템(AI)이 스스로 논문을 번역해서 씹어먹은 뒤, "아하, 이건 '최신 프롬프트 기법' 관련이구나!"라고 판단하여 마인드맵에 `다중 문맥 이해 (Multi-Contextual)` 이라는 친절한 한글 노드를 알아서 달아주고 한글 요약본을 정리해 놓습니다.
- **우선순위**: **필수 (P0)** 

### [기능 2] 인터랙티브 자가 성장형(유기적) 마인드맵 (Organic Dynamic Mindmap)
- **설명**: 사전에 하드코딩된 고정 카테고리에 얽매이지 않습니다. 데이터가 쌓이거나 완전히 새로운 질문/정보가 들어올 경우, AI가 스스로 컨텍스트를 파악해 맵을 재구조화(Clustering) 하거나 실시간으로 **새로운 분류(Root Node)**를 창조해 냅니다. 상황과 유저의 질문에 맞게 다이내믹하게 유기적으로 성장하고 구조를 바꾸는 지식의 숲입니다.
- **사용자 시나리오**: 기존 맵에 아예 없던 도메인인 "요즘 AI 비디오 툴방식들 좀 정리해줘"라고 입력하면, 캔버스의 구석 텅 빈 공간에서 "🎥 AI 비디오 툴"이라는 거대한 새로운 뿌리가 애니메이션과 함께 솟아오른 뒤 하위 갈래들이 방사형으로 꽂히며 새로운 세계가 스스로 구축됩니다.
- **우선순위**: **필수 (P0)** 

### [기능 3] JARVIS 딥 컨텍스트 챗봇 (RAG 기반 사이드바)
- **설명**: 대시보드 우측을 항시 차지하는 비서. 사용자가 마인드맵에서 클릭 중인 노드를 컨텍스트로 인지하며, 내장된 DB 검색 기반으로 응답(RAG). 유기적으로 변하는 맵의 현 상황을 파악해 대화.
- **사용자 시나리오**: 맵에서 특정 용어를 클릭한 후 "방금 그 뜻이 정확히 뭐야? 나는 코딩 1도 몰라."라고 질문하면, 관련 내부 스크랩 문서를 찾아와 "API는 식당의 웨이터 같은 거예요~" 라며 초보자 맞춤 찰떡 비유로 요약해 줌.
- **우선순위**: **필수 (P0)** 

### [기능 4] 옴니 서치(Omni-Search) 및 하이라이트
- **설명**: 화면 좌상단의 검색창에 용어/개념/의미를 입력 시, 마인드맵 상의 관련된 노드들에 초고속 줌인되어 즉시 포커싱/이동 후 하이라이트 제공.
- **사용자 시나리오**: "로그 보관" 이라고 검색해 엔터를 치면, 앱 캔버스가 빠르게 스크롤되며 관련 브랜치로 딱 멈춘 뒤 관련된 지식 노드 3개를 하이라이트(Glow) 처리함.
- **우선순위**: **필수 (P0)** 

### [기능 5] (선택) 해커뉴스 스타일 데일리 피드
- **설명**: 영문 해외 주요 AI 정보를 정기적으로 크롤링하여, 한글 번역 후 극도로 깔끔한 '순수 텍스트 리스트(HackerNews 뷰)' 형식으로 노출함.
- **우선순위**: **선택 (P1 - 1차 기능 완료 후 배포 전 고려)**
