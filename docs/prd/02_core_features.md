# 2. 핵심 기능 명세

[<- 마스터 인덱스로 돌아가기](./index.md)

이 시스템의 뼈대를 이루는 4가지 핵심 MVP 기능 명세입니다.

### [기능 1] 매직 날리지 주입기 (Magic Ingestion & Draft Verification)
- **설명**: 텍스트, URL 덤프 시 백그라운드 AI 파서가 작동합니다. **영어 등 외국어라도 100% 한글로 요약/번역**합니다. 무작정 맵에 섞어 얽히게 만들지 않고, 팝업(Draft Modal) 기능을 통해 AI의 요약 결과와 **[출처의 신뢰도/정확성]**을 사용자에게 먼저 보여줍니다. 사용자가 눈으로 확인하고 '승인(Approve)'할 때만 맵에 영구 편입되어 지식 오염을 차단합니다.
- **사용자 시나리오**: 복잡한 해외 AI 논문을 옴니바에 넣고 "주입"을 누르면, 화면에 팝업창이 떠서 "이 정보는 공식 리서치 논문을 바탕으로 한 신뢰할 만한 정보입니다"라는 코멘트와 요약본을 보여주며, 승인을 클릭하면 맵의 빈 공간에 가지를 뻗어 안착합니다.
- **우선순위**: **필수 (P0)** 

### [기능 2] 인터랙티브 유기적 마인드맵 
- **설명**: 고정 카테고리에 얽매이지 않고, 새로운 질문에 맞게 AI가 맵을 재구조화 하거나 실시간으로 새로운 분류(Root Node) 및 서브 노드 가지들을 스스로 뻗어내며 방사형 지식 트리를 구축합니다.
- **우선순위**: **필수 (P0)** 

### [기능 3] JARVIS 딥 컨텍스트 챗봇 (포커스 RAG)
- **설명**: 대시보드 우측의 RAG 비서. 단순히 맵 전체를 넓게 아는 게 아니라, **사용자가 마인드맵에서 현재 클릭 중인 노드를 컨텍스트 최우선순위로 인지**해 더욱 깊게 대화합니다.
- **우선순위**: **필수 (P0)** 

### [기능 4] 옴니 서치(Omni-Search) 및 하이라이트
- **설명**: 화면 좌상단 토글 바를 `[검색]` 모드로 맞춘 뒤 키워드를 치면, 새로운 걸 만들지 않고 맵 상의 가장 연관 깊은 장소로 카메라가 초고속 로밍(Pan/Zoom)한 뒤 형광 효과(Glow)로 강조해줍니다.
- **우선순위**: **필수 (P0)** 

### [기능 5] (NEW) 지식 타임머신 (버전 스냅샷 관리)
- **설명**: 마인드맵이 무분별한 주입으로 엉망으로 얽혔을 때를 대비한 **지식 버전 관리(Version Control)** 시스템. 맵의 전체 구조 상태를 스냅샷 백업해 둡니다.
- **사용자 시나리오**: 잘못된 정보를 잔뜩 넣어 맵이 난잡해졌을 때, 버전 관리 버튼을 눌러 "오늘 오전 깔끔했던 시점"으로 다시 롤백(Revert)하여 지식의 순수성을 되찾습니다.
- **우선순위**: **필수 (P0 - 지식 오염 보호 시스템)**
